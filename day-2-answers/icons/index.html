Ícones

Uma das práticas mais comuns hoje é utilizar SVG como fonte de ícones.

Uso direto do SVG

A primeira forma mais simples é colocar o conteúdo do SVG direto no lugar onde se deseja renderizar o ícone.

Este é um ícone 

  <svg class="icon" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 448 512">
    <path d="M320 448v40c0 13.255-10.745 24-24 24H24c-13.255 0-24-10.745-24-24V120c0-13.255 10.745-24" />
  </svg>

Uso referenciado do SVG

Uma segunda forma, para reaproveitar o mesmo SVG, é definindo símboloes e referenciando-os.

Este é outro ícone 

  <svg
    class="icon"
    xmlns="http://www.w3.org/2000/svg"
    viewBox="0 0 448 512"
  >
    <use xlink:href="#ic-bars" />
  </svg>

  <svg class="hidden">
    <symbol id="ic-bars">
      <path
        d="M320 448v40c0 13.255-10.745 24-24 24H24c-13.255 0-24-10.745-24-24V120c0-13.255 10.745-24"
      />
    </symbol>
  </svg>

Via CSS

Uma terceira forma é utilizando o pseudo-elemento :hover e usando como o SVG como background

Um terceiro exemplo 

  <span class="icon icon-home" aria-hidden="true">Um terceiro exemplo</span>
